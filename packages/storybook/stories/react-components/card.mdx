# Card

The Card component is a versatile container used to group related content like text, images, and actions in a styled, organized layout.

<ArgTypes of={Stories} />

## Usage

```js
import { Card } from '@uoguelph/react-components';
```

### Basic

All cards must contain a title at minimum.

<Canvas of={Stories.Basic} sourceState="shown" />

### Card Body Content

The card can contain additional body content.

<Canvas of={Stories.WithBodyContent} sourceState="shown" />

### Card Image

An image can be added to the card, providing more visual interest.

<Canvas of={Stories.WithImage} sourceState="shown" />

### Card Footer

The card footer is a section that appears at the bottom of the Card, use it for extra info or actions.

<Canvas of={Stories.WithFooter} sourceState="shown" />

The card element can be used as a hyperlink to another page.
